艺术成就 · 璀璨星光

国家代表

参与联合国世界中文日，代表国家献唱《和平颂》等作品，被主流媒体誉为"文化输出典范"。作为优秀艺术家代表，参与文艺工作者座谈会。

音乐奖项

11年间荣获超过186项国内外重量级音乐奖项，包括年度歌手、最佳男歌手、年度创作歌手、国家级"五个一工程"奖、全球华语音乐榜中榜等。

影视金曲

为超过80部影视作品演唱主题曲/插曲，享有"OST之王"美誉，代表作如《大鱼》《小美满》等。致力于推动影视音乐向"艺术品"高度升级。

专辑续航力

《反深代词》成绩亮眼，成为限购政策后内地歌手专辑销量TOP1。首位凭数字专辑上榜IFPI国际唱片业协会2024年"全球畅销专辑榜"的内地歌手。

演出里程碑

多次参与央视春晚等国家级重大演出，现场表现力广受赞誉。已举办超过50场个人演唱会，是内地最年轻开启体育场级别巡演的男歌手，足迹横跨五国四大洲。

国际影响力

在YouTube平台累计播放量超1亿，多次登上Billboard中国榜，多语种演唱作品实现跨国传播。被授予"COP15公益宣传大使"称号，积极带动环保行动。
